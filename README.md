Thank you for viewing my second side project:
"A Comparative Evaluation of Supervised Machine Learning Models for Causal Gene Recovery in Autism Spectrum Disorder"

The files that you will find above are...
1. .R coding file.
2. .csv simulated dataset file.

This project pushed the limits of my coding ability, and was a GREAT introduction to machine learning in a scientific, variable selection context. The skills I believe I learned in this project include:
1. Tuning various machine learning models, specifically PLR, RF, XGB and SVM models.
2. Using PLR, RF, XGB and SVM models to determine the importance of 250 features, by permutation importance or gain, in relation to a specific disease.
3. Creating visualizations for the end-products of these ML models.
4. Improving coding proficiency and efficiency.
